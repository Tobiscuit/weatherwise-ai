# WeatherWise AI

A smart weather forecasting application providing on-demand reports and witty, AI-generated summaries for any location. Built with TypeScript, Node.js, and Google Cloud Run.

## Features

-   **On-Demand Weather:** Get the current temperature, conditions, and high/low forecasts for any city.
-   **AI-Powered Summaries:** Receives a unique, witty weather summary generated by Google's Gemini AI.
-   **Dynamic Weather Icons:** The interface displays custom icons based on the current weather conditions.
-   **Serverless Architecture:** Built on a modern, scalable serverless backend using Fastify on Google Cloud Run.
-   **Optimized Caching:** Implements an intelligent caching layer to provide near-instant results for recent searches.

## Technologies Used

-   **Frontend:** HTML, Nunjucks, TailwindCSS
-   **Backend:** Node.js, Fastify, TypeScript
-   **API:** Google Gemini for AI summaries
-   **Deployment:** Docker, Google Cloud Run, Google Cloud Build

## Local Development

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/YourUsername/weatherwise-ai.git
    cd weatherwise-ai
    ```

2.  **Install dependencies:**
    This project uses `pnpm` for package management.
    ```bash
    pnpm install
    ```

3.  **Authenticate for Local Development:**
    For local development, the application uses Application Default Credentials. Authenticate by running:
    ```bash
    gcloud auth application-default login
    ```

4.  **Create an environment file:**
    Create a `.env` file in the root of the project and add your Cloud Run service URL.
    ```
    CLOUD_RUN_URL=your_cloud_run_url_here
    ```

5.  **Run the development server:**
    This command starts the backend server, TailwindCSS watcher, and live reload all at once.
    ```bash
    pnpm dev
    ```

## Deployment

This project is configured for continuous deployment to Google Cloud Run via Google Cloud Build. Pushing to the `main` branch will automatically trigger a new build and deploy the service.

Secrets (like API keys) are managed securely using Google Secret Manager. See `DEPLOYMENT_GUIDE.md` for detailed instructions.
